# 介绍
忽然中二病发作想到做一个符文字体，可以作为装饰性元素以及超酷的纹饰
但是只做了英文字母以及四声还有一些标点符号。具体字符详见文件夹里的图片
数字采用12进制，10和11分别是#和$

主要支持 : a-z A-Z 0-9+#$ 四声：（一声：< 二声：> 三声：^ 四声：\）

大伙觉得像异世界的文字所以就叫它IsekaiType了
